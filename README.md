# Employee Attrition Analysis & Retention Strategy

## 📊 Project Overview
This project analyzes employee attrition using HR data to identify key factors influencing employee turnover.

## 🛠 Tools Used
- Excel (Data Cleaning)
- Power BI (Dashboard & Visualization)

## 📌 Key Insights
- Higher attrition observed in employees aged 18–26
- Low-income group shows significantly higher attrition
- Certain job roles (e.g., Laboratory Technician, Sales) have higher attrition
- Early tenure (0–2 years) shows maximum employee exits

## 💡 Business Recommendations
- Improve salary structure for low-income employees
- Strengthen onboarding and early-career engagement
- Provide clear career growth opportunities
- Implement role-specific retention strategies

## 🎯 Outcome
The project helps organizations identify attrition drivers and take data-driven actions to improve employee retention.
